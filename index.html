Keyboard keys

I created a CSS class to style some letters as if they were keyboard keys. Press the keys to see the pressing effect.

The combination Ctrl + Alt + Del is one of the most common key combinations for intermediate and advanced Windows users.

To achieve the pressing effect was especially trick.

RGB Gradient Effect

The gradient effect below is made by using pure CSS.

The CSS code is shown below:

#rgb-gradient {
background:
linear-gradient(90deg, red, green, blue);
display: block;
width: 100%;
height: 50px;
}
